-The application must be delivered to the client no later than the 4th December 2014
-The application must be compatible with Android devices running Android 4.1.1 and above
-The application must be compatible with a variety of screen sizes, and scale all elements accordingly.
-The application colour scheme must match that of the Cadence health website
-The application must follow the Android Design guidelines and conform to common UI patterns found at

	-The application does not redefine the expected function of a system icon (such as the "home" or "back " buttons)

-The application must request the absolute minimum permissions that it needs to operate normally.
